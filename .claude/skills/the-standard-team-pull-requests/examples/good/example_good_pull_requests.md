# ✅ Good Examples — Pull Requests

## PR Titles

```
FOUNDATIONS: Add Student
BROKERS: Insert Student
BROKERS: Select Student By Id
CONTROLLERS: POST Student
DATA: Add Student Model
CONFIG: Add Storage Connection String Settings
DOCUMENTATION: Update Readme With Setup Instructions
```

## PR Description with Issue Auto-Close

```
This PR adds the Student foundation service with full validation and exception handling.

Closes #42
```

## PR Description Closing Multiple Issues

```
This PR adds the Student broker and foundation service.

Closes #10, closes #23
```

## PR Description Linking Without Auto-Close

```
This PR updates the storage broker.

Related: #15
```
