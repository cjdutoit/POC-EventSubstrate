# The Standard C# — Files — Checklist

- [ ] File name uses PascalCase (tsc-csharp-files-001)
- [ ] File name ends with `.cs` (tsc-csharp-files-001)
- [ ] File name contains no underscores, hyphens, or all-lowercase (tsc-csharp-files-002)
- [ ] Partial class files follow `{RootClass}.{Aspect}.cs` pattern (tsc-csharp-files-003)
- [ ] Nested partial files follow `{RootClass}.{Aspect}.{SubAspect}.cs` pattern (tsc-csharp-files-004)
