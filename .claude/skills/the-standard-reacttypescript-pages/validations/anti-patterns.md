# Anti-Patterns
## Business Logic in Page
**Violates:** tsr-pages-007
Page contains validation, calculations, or business rules.
## Data Fetching in Page
**Violates:** tsr-pages-008
Page fetches data using fetch/axios instead of hook.
## Duplicate Rendering
**Violates:** tsr-pages-004, tsr-pages-015
Page duplicates component rendering logic instead of composing.
