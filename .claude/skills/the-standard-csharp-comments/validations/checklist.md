# The Standard C# — Comments — Checklist

- [ ] No comment restates what the code already communicates (tsc-csharp-comments-001, tsc-csharp-comments-006)
- [ ] Copyright block uses `//` single-line dashed-border style (tsc-csharp-comments-002)
- [ ] Copyright block has exactly two content lines: copyright holder and license note (tsc-csharp-comments-002)
- [ ] No `/* */` block comment style is used (tsc-csharp-comments-003)
- [ ] No XML `<copyright>` tag is used (tsc-csharp-comments-004)
- [ ] Complex or inaccessible methods include: Purposing, Incomes, Outcomes, Side Effects documentation (tsc-csharp-comments-005)
