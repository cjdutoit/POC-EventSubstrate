# The Standard React TypeScript — Components — Checklist

- [ ] Component is pure rendering function (tsr-comp-001)
- [ ] All data received via props (tsr-comp-002)
- [ ] Props typed with TypeScript (tsr-comp-003)
- [ ] In correct directory (tsr-comp-004)
- [ ] File name follows pattern (tsr-comp-005)
- [ ] Named export used (tsr-comp-006)
- [ ] Loading/error via props (tsr-comp-007)
- [ ] No service/broker/API calls (tsr-comp-008)
- [ ] No business logic (tsr-comp-009)
- [ ] No useEffect data fetching (tsr-comp-010)
- [ ] Function component syntax (tsr-comp-011)
- [ ] Props destructured (tsr-comp-012)
- [ ] Under 200 lines (tsr-comp-013)
- [ ] Single responsibility (tsr-comp-014)
- [ ] Complex handlers via props (tsr-comp-015)
- [ ] Props type in models/ (tsr-comp-016)
- [ ] Optional props use ? (tsr-comp-017)
- [ ] Callbacks optional appropriately (tsr-comp-018)
- [ ] No foundation models in props (tsr-comp-019)
- [ ] View models used (tsr-comp-020)
