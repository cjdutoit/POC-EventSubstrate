| `fix` | tst-commits-008 — vague description |
| `update student` | tst-commits-004, tst-commits-008 — no category, vague description |
| `BROKERS: insert student` | tst-commits-006 — description not Pascal Case |
| `VIEWS: Add Student` | tst-commits-003, tst-commits-009 — TDD category must use FAIL/PASS format |
| `ShouldAddStudentAsync` | tst-commits-001 — missing -> FAIL or -> PASS suffix |
| `wip` | tst-commits-008 — forbidden vague description |
