# The Standard C# — Directives — Checklist

- [ ] All `using` directives are at the top of the file, outside any namespace block (tsc-csharp-directives-001)
- [ ] Directives are grouped: System → Microsoft → third-party → project-local (tsc-csharp-directives-002)
- [ ] Each group is separated by exactly one blank line (tsc-csharp-directives-003)
- [ ] Directives within each group are sorted alphabetically (tsc-csharp-directives-004)
- [ ] No unused `using` directives remain in the file (tsc-csharp-directives-005)
