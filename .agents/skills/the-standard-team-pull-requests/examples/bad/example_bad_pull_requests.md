# ❌ Bad Examples — Pull Requests

## Bad PR Titles

| PR Title | Violation |
|---|---|
| `foundations: Add Student` | tst-pull-requests-008 — lowercase category |
| `Foundations: Add Student` | tst-pull-requests-008 — mixed-case category |
| `fix` | tst-pull-requests-009 — vague description, missing category |
| `update student model` | tst-pull-requests-001, tst-pull-requests-009 — no category format, vague |
| `HELPERS: Add Utility` | tst-pull-requests-010 — HELPERS is not an approved category |
| `FOUNDATIONS: add student` | tst-pull-requests-003 — description not Pascal Case |
| `MISC: Various Changes` | tst-pull-requests-009, tst-pull-requests-010 — vague + invalid category |
