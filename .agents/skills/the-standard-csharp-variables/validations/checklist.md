# The Standard C# — Variables — Checklist

- [ ] No single-letter or abbreviated variable names (tsc-csharp-variables-001)
- [ ] Collection names are plural without type suffix (tsc-csharp-variables-002)
- [ ] No type suffix or prefix in variable names (tsc-csharp-variables-003)
- [ ] Null/default variables are prefixed with `no` (tsc-csharp-variables-004)
- [ ] `var` is used when right-hand side type is clear (tsc-csharp-variables-005)
- [ ] Explicit type is used for method-return semi-clear types (tsc-csharp-variables-006)
- [ ] Single-property types use declare-then-assign style, not initializer block (tsc-csharp-variables-008)
- [ ] Multi-property types use object initializer block (tsc-csharp-variables-009)
- [ ] Declarations over 120 chars break at `=` sign (tsc-csharp-variables-010)
- [ ] Single-line declarations are stacked with no blank lines between them (tsc-csharp-variables-011)
- [ ] Multi-line declarations are surrounded by blank lines (tsc-csharp-variables-012)
